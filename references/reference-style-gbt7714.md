# 参考文献著录格式（GB/T 7714）

本文件规范**文末参考文献条目本身**的著录格式。正文引用标记的上标与 Word `REF` 交叉引用实现见 `references/citation-crossrefs-ooxml.md`；二者配合使用。

默认采用 GB/T 7714《信息与参考文献著录规则》的**顺序编码制**：参考文献按正文首次引用先后编号，文末列表用 `[1]`、`[2]`… 顺序排列。学校/期刊另有要求（如著者—出版年制、GB/T 7714-2015 的“双语 / DOI”要求、或 APA/IEEE 等）时以其模板为准，并在交付说明中注明。

## 文献类型标识

条目题名后用方括号标注文献类型（必要时叠加载体类型）：

| 标识 | 文献类型 |
| --- | --- |
| `[M]` | 专著（图书） |
| `[J]` | 期刊文章 |
| `[D]` | 学位论文 |
| `[C]` | 会议论文集 / 会议录 |
| `[C]//` | 析出的会议论文 |
| `[S]` | 标准 |
| `[P]` | 专利 |
| `[R]` | 报告 |
| `[N]` | 报纸文章 |
| `[EB/OL]` | 电子资源（网络） |
| `[DB/OL]` | 联机数据库 |

## 各类型条目格式与示例

- 期刊 `[J]`：主要责任者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
  - 示例：王伟, 李明. 梯级水库优化调度研究[J]. 水利学报, 2020, 51(3): 312-320.
- 专著 `[M]`：主要责任者. 书名[M]. 版本(第 1 版可省). 出版地: 出版者, 出版年: 起止页码.
  - 示例：张华. 水利计算原理[M]. 2 版. 北京: 中国水利水电出版社, 2018: 45-60.
- 学位论文 `[D]`：作者. 题名[D]. 保存地: 保存单位, 年.
  - 示例：陈强. 水库防洪调度风险分析[D]. 武汉: 武汉大学, 2019.
- 会议论文 `[C]`：作者. 题名[C]//论文集编者. 论文集名. 出版地: 出版者, 出版年: 页码.
- 标准 `[S]`：标准编号, 标准名称[S]. 出版地: 出版者, 出版年.
  - 示例：GB/T 7714-2015, 信息与文献 参考文献著录规则[S]. 北京: 中国标准出版社, 2015.
- 电子资源 `[EB/OL]`：作者. 题名[EB/OL]. (发布日期)[引用日期]. 访问地址.

## 著录要点

- 作者：1–3 人全部列出，姓名之间用逗号；超过 3 人时只列前 3 人，其后加“等”（英文文献用“et al.”）。
- 西文作者姓前名后，名可缩写（如 Smith J A）；中文作者姓名照写。
- 题名后紧跟文献类型标识；刊名/出版者按原文，不随意翻译。
- 卷(期)、页码、年份齐全；期刊给出年、卷(期)、起止页。
- 各项之间的标点（点、逗号、冒号、斜杠）按上面的体例统一，全表一致。
- 条目用悬挂缩进（首行顶格、回行缩进对齐编号后正文），字号默认五号。
- 中外文文献可分列或统一编号，但同一论文体例一致。

## 与正文引用配合

- 文末每条参考文献的编号建立稳定书签 `ref_001`、`ref_002`…（见 citation-crossrefs 文件）。
- 正文引用 `[1]` 等做成上标 + `REF` 字段，指向对应条目，保证“正文引用号”与“文末编号”一致并可自动更新。
- 顺序编码制下，若调整参考文献顺序，需重排编号、重建书签并更新全部域。

## 交付前检查

- 文献类型标识齐全且正确（`[J]/[M]/[D]/[C]/[S]/[EB/OL]` 等）。
- 作者、题名、出版信息、年卷期页齐全；超过 3 人是否用“等 / et al.”。
- 全表标点体例一致；悬挂缩进、字号一致。
- 正文每个引用号都能在文末找到对应条目，无缺号、无凭空编号。
