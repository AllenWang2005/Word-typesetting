# 文档结构与页面设置

适用于正式中文学位论文 / 课程设计 / 工程报告。覆盖文档整体结构、页面设置、页眉页脚、页码与分节、目录、字号体系、多级编号与孤行控制。学校或期刊模板与本文冲突时，以模板为准，并在交付说明中注明依据。

## 文档结构与顺序

正式论文/报告的典型顺序（按需取舍，但顺序应一致）：

1. 封面（题目、作者、学号、专业、指导教师、日期、单位/学校）。
2. 原创性声明 / 授权使用声明（学位论文常需）。
3. 中文摘要 + 关键词（3–5 个，关键词之间用分号或逗号，全文统一）。
4. 英文摘要 Abstract + Keywords（与中文摘要内容对应）。
5. 目录（必要时附图目录、表目录）。
6. 正文（含引言/绪论、各章、结论）。
7. 参考文献。
8. 附录（含附录代码、数据表等）。
9. 致谢。

- 各主要部分（摘要、目录、各章、参考文献、附录、致谢）建议另起页；学位论文常要求“章”另起页，必要时另起奇数页。

## 页面设置

- 纸张：A4（210 mm × 297 mm），纵向为主，宽表可用横向页。
- 页边距：按学校模板；无模板时常用上下各 2.5–3.0 cm、左右各 2.5–3.0 cm。
- 装订线（gutter）：需要打印装订时预留装订线（如左侧或上侧 0.5–1.0 cm），避免正文被装订边遮住。

## 页眉页脚

- 页眉：可放章节名或学校/论文名；页眉下常有一条横线。
- 页眉页脚可设“奇偶页不同”：奇数页页眉显示章标题、偶数页显示论文题目（按模板要求）。
- 正文与前置部分（摘要、目录）通常分属不同节，页眉内容可不同。

## 页码与分节

- 用 Word 分节符（Section Break）划分前置部分与正文，使页码体系可以切换。
- 前置部分（摘要、目录等）用罗马数字页码（Ⅰ、Ⅱ、Ⅲ…）。
- 正文从第 1 页起用阿拉伯数字页码（1、2、3…），并在分节处“重新起始编号”。
- 页码位置（页脚居中或外侧）全文一致。

## 目录

- 目录由标题样式自动生成（引用 Word 标题样式 / 大纲级别），不要手工敲页码。
- 层级一般到三级标题（1 / 1.1 / 1.1.1）。
- 目录条目用制表位前导符（tab leader）点线连接标题与页码，页码右对齐。
- 需要时附“图目录”“表目录”，由题注自动生成。
- 章节、页码调整后必须更新目录域（整篇更新域）。

## 字号体系

各结构层级的默认字号见 `references/formatting-standard.md` 的“字号层级表”。要点：

- 题目二号黑体居中；一级标题三号/小三、二级四号、三级小四，均黑体加粗。
- 正文小四（12 pt），中文宋体、西文与数字 Times New Roman。
- 参考文献条目五号、脚注小五。
- 学校/期刊模板优先于以上默认值。

## 多级编号自动化

- 章节编号用绑定到标题样式的 Word 多级列表实现自动编号（`1` / `1.1` / `1.1.1`），不要手动键入数字。
- 图、表、公式编号用 Word 题注 / `SEQ` 域按章自动编号（图 1-1、表 2-3、式(1-1)），便于交叉引用与自动更新。
- 调整顺序后，编号与正文交叉引用应能一次性更新域后保持一致。

## 孤行控制

- 标题不能孤立在页面最底部：其后至少应有一行正文，或与下一段同页。
- 正文段落开启“孤行控制 / 段中不分页”，避免段落首行或末行单独跨页（widow/orphan）。
- 图与图题、表与表题尽量同页；表格跨页时重复表头，不允许从中间硬断。
